A Not-So Oceanic Voyage (Adventure.zip)

Walthrough

Talk to the captain to get the map

Use the talkto command. After using it once, you can use continue/yes/no. Remember to type the name of the addressee the first time you address them.

Talk until the NPC says they have nothing else to say.

The help command lists all of the available commands

To see the map, type use map.

Getting the map is technically optional but in practice not.

Talk to the engineer to get the radio

Talk until the NPC says they have nothing else to say.

Using the radio tells the ship you're on.

Getting the radio is technically optional but in practice not.

Walk to the center of Ship F to get the sail

The get command picks it up

Move the ships so that Ship C can be moved left

The sail command, followed by a cardinal direction, does this.

You have to be on the ship to operate it.

move F east.

move B west.

move D north.

move E east.

move A two spaces south.

After moving C west, you win!
